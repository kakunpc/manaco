## 動作・ビルドに関する注意
- MANACOの処理はNDMFビルド時に実行されます。Editor上の見た目と最終ビルド結果が異なる場合は、NDMF Previewまたはテストビルドで確認してください。
- 目領域UVが未設定、または対象ポリゴンが見つからないEyeRegionはスキップされます。
- `targetRenderer` / `sourceRenderer` / `customMaterial` の未設定項目があると、その領域は適用されません。

## 設定に関する注意
- `CopyEyeFromAvatar` モードでは、コピー元アバター（Prefab）とコピー元プリセットの整合が必要です。
- プリセットはレンダラー名を参照するため、モデル側のオブジェクト名変更で再マッピングが必要になる場合があります。
- マテリアル定義は目ごとに分離して設定する想定です（左目・右目・瞳孔）。

## フォールバックテクスチャ
- ビルド時に `_MainTex` が未設定のマテリアルには、フォールバックテクスチャ生成が行われます（設定有効時）。
- `_MainTex` を持たないシェーダーではベイク処理はスキップされます。

## 配布に関する注意
- `ManacoPreset` / `Manaco Material Definition` は `.asset` 配布が可能です。
- MANACOで利用しやすくするために `ManacoPreset` や `Manaco Material Definition` を同梱するのは自由に行えます。ライセンス表記や連絡も不要です。
- `ManacoPreset` にはアバターの目のUV座標が記録されています。配布に関してはアバターの利用規約を必ず確認の上、アバターの利用規約に従ってください。`ManacoPreset` で発生した問題に関しては一切の責任を負いかねます。
