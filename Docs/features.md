## MANACOとは
MANACO（Material Assign Non-destructive Assistant for Customization Operations）は、VRChatアバターの目のマテリアル再割り当てを非破壊で行うためのUnity Editor拡張です。  
NDMF（Modular Avatar）ビルドフローに統合され、元メッシュを直接書き換えずに目領域へマテリアル適用や目テクスチャ抽出を行えます。

## 主な機能

### 1. 目領域への非破壊マテリアル割り当て
- `マテリアル割り当て` モードで、左目・右目（必要に応じて瞳孔）に個別マテリアルを割り当てます。
- UV Islandを使って目ポリゴンを指定し、ビルド時に専用SubMeshへ分離します。
- 元メッシュは複製して処理するため、編集時の破壊的変更を回避できます。

### 2. 別アバターから目テクスチャをコピー
- `別アバターの目をコピー` モードでコピー元アバターを参照します。
- コピー元の目UV領域を抽出し、コピー先の目へ反映します。
- アバター改変の移植作業を効率化できます。

### 3. プリセット運用
- `ManacoPreset` により、アバター別の目領域定義を再利用できます。
- `Manaco Material Definition` でマテリアル定義をアセット化して配布できます。
- アセット配布ベースのため、配布先で扱いやすい構成です。

### 4. UVエディタ
- テクスチャプレビュー上でUV Islandを選択できます。
- 左クリックで追加し、右クリックで削除できます。
- 選択領域をSceneView側でも確認しながら調整できます。

### 5. NDMF統合
- NDMFのGeneratingフェーズで処理を実行します。
- Modular Avatar処理後に適用される構成です。
- NDMF Previewと連携できます（設定で有効化可能です）。

## 想定ユースケース
- 既存アバターに目シェーダーを導入したい場合に向いています。
- 別アバターの目表現をコピーして再利用したい場合に向いています。
